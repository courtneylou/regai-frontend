# REGAI – Requirements Elicitation Game

REGAI is a learning tool for practicing requirements gathering through scenario-based conversations with simulated (AI/LLM) stakeholders.

This repo contains the frontend UI only that can later be used with a backend and AI.

---

### What’s in this repo

#### Implemented
- UI for:
  - Login / signup (mocked)
  - Scenario selection
  - Stakeholder chat interface
  - Requirements capture panel
  - Account page
- Component system - shadcn/ui
- Style and theme - Tailwind

#### Not implemented yet
- Real authentication
- Database
- AI responses
- Backend services

---

## Tech stack

- React 18
- TypeScript
- Vite
- Tailwind CSS (v3)
- shadcn/ui (https://ui.shadcn.com/)
- Radix UI (https://www.radix-ui.com/)
- TanStack Query
- React Router
- Lucide Icons (https://lucide.dev/icons/)
